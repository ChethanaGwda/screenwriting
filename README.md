# ✍ Air Writing using Hand Gesture

## 📌 Project Description
This project allows the user to **write on screen using hand gestures**.
Using computer vision, the program detects the **index finger movement** and draws on the screen in real time.

The project uses **MediaPipe** and **OpenCV** to track hand landmarks and create a virtual drawing board.

---

## 🚀 Features
- Write in air using **index finger**
- **Change pen color** using buttons
- **Clear screen** button
- Real-time hand tracking
- Easy to use interface

---

## 🛠 Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
